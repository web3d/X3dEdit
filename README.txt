X3D-Edit version 3.2 was upgraded to version 3.3 in 2012
to match the updated version of the X3D Specification.
A number of features were added, but there were no
fundamental changes to the underlying X3D-Edit codebase.

Please see the X3dEdit3.3 codebase for the updated source.

Project home page:  https://savage.nps.edu/X3D-Edit
